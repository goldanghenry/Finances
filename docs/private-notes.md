# 비공개 노트 (로컬 전용)

`private/` 폴더는 **`.gitignore`에 등록**되어 있어 Git·GitHub Pages에 **올라가지 않습니다.**

## 설정

저장소 루트에서:

```bash
mkdir -p private
```

## 예시 파일 (본인만 생성)

| 파일 | 용도 |
|------|------|
| `private/my-m.md` | 실제 **M**, **R**, **B**, **T** 등 본인 숫자 |
| `private/bucket-checklist.md` | [STUDY-START](../00-roadmap/STUDY-START.md) Day 1 산출물 |
| `private/weekly-macro.md` | 주간 거시 체크 기록 |
| `private/reading-log.md` | 필독·L4 진도 |

공개 교재에는 [DEPTH-STANDARD](DEPTH-STANDARD.md) §교육용 금액 기호(**M**·α)만 쓰고, **구체 금액은 `private/`에만** 두세요.
