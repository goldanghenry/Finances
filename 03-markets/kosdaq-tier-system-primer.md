# 코스닥 승강제 5분 요약 — L1

## TL;DR

- 코스닥을 **프리미엄(약 100)·스탠더드·관리군** 으로 나누는 **승강제** 도입 **추진** (보도상 **2026년 10월** 시행 목표).
- **프리미엄** = 우량·기관·ETF 자금 기대 / **관리군** = 퇴출·리스크 집중. 관리군은 상장폐지 통로로 이해.
- **동전주·시총·적자** 퇴출 규칙도 **2026.7~** 강화 — 승강제와 **겹침**.
- 핵심은: K리그 승강제와 비슷하지만, 관리군은 단순 강등이 아니라 **퇴출 위험** 구간. 주식이 상장폐지되면 거래 자체가 불가능해진다.
- 개별 코스닥 **테마·소형** → [Bucket 4](../04-portfolio/time-horizon-and-buckets.md) · 코어는 **지수·ETF**.
- 실제 투자에서는: 보유 코스닥 종목이 어느 세그먼트로 분류될지 재무 기준(시총·자본잠식)을 미리 점검할 것.
